The Algorithm

Preprocessing

  • Choose L functions gj, j = 1,…L, by setting gj = (h1,j, h2,j,…hk,j), where h1,j,…hk,j are chosen at random from the LSH family H
  • Construct L hash tables, where, for each j = 1,…L, the jth hash table contains the dataset points hashed using the function gj

Query for point q

For each j = 1, 2,…L

  • Retrieve the points from the bucket gj(q) in the jth hash table
  • For each of the retrieved point, compute the distance from q to it, and report the point if it is a correct answer
  • (optional) Stop as soon as the number of reported points is more than L
